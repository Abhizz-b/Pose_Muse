#Pose Muse

> *People often get confused about what pose to strike — Pose Muse solves that!*

An AI-powered Flutter app that detects your body using your camera and recommends the perfect poses for you. No more confusion, just confidence. 📸✨

---

## 🚧 Status

> **Under Development** — This project is actively being built. Features may be incomplete or subject to change.

---

## ✨ Features

- 📷 **Camera-based Body Detection** — Real-time body detection using your device camera
- 🤸 **Pose Recommendations** — AI suggests the best poses based on your body position
- 🎨 **Clean UI** — Minimal and intuitive interface designed for ease of use
- 🤖 **AI Integration** *(coming soon)* — Smarter pose analysis powered by machine learning

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Android / iOS device or emulator

### Installation

```bash
# Clone the repo
git clone https://github.com/TERA_USERNAME/Pose_Muse.git

# Go into the folder
cd Pose_Muse

# Install dependencies
flutter pub get

# Run the app
flutter run
```
